# 🎧 Malik Music Player

A modern, premium-looking web-based media player built for smooth audio and video playback with a futuristic **Blue Neon UI**.
Malik Music Player delivers a clean, responsive, and immersive media experience inspired by modern desktop players.

---

# ✨ Features

## 🎵 Media Support

* Play `.mp3` audio files
* Play `.mp4` video files
* Smooth HD playback
* Drag & Drop media support

---

## 🎬 Video Features

* Fullscreen mode
* Mini screen mode
* Responsive video player
* Modern playback controls

---

## 🔤 Subtitle Support

* Upload subtitle files
* Supports:

  * `.srt`
  * `.vtt`
* Enable/Disable subtitles
* Beautiful subtitle overlay

---

## 🔊 Audio Controls

* Increase/Decrease volume
* Mute/Unmute
* Smooth volume slider
* Real-time audio adjustments

---

## 🖱️ Drag & Drop

Simply drag media files into the player window and start playing instantly.

---

## 🎨 Premium UI Design

* Futuristic Blue Neon Theme
* Glassmorphism effects
* Smooth animations
* Responsive layout
* Modern hover effects
* Professional alignment and spacing

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Tailwind CSS
* Framer Motion

## Backend

* Python Flask / FastAPI

## Media Processing

* HTML5 Media APIs
* FFmpeg (optional)

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Malik231-spec/Music_Player.git
cd Music_Player
```

---

## 2️⃣ Install Frontend Dependencies

```bash
npm install
```

---

## 3️⃣ Start Frontend

```bash
npm run dev
```

---

## 4️⃣ Start Backend

```bash
python app.py
```
